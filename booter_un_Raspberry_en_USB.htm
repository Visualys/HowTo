Sur la carte SD d'origine,
Ajouter au fichier boot/config.txt la ligne suivante :
program_usb_boot_mode=1
Redemarrer avec le Raspberry (toujour avec la carte SD pour le moment).
Vérifier que le Raspberry a bien pris en compte la modification : 
$ vcgencmd otp_dump | grep 17
17:3020000a => la réponse doit être cette valeur, sinon il est inutile de continuer

Ensuite le demarrage sur USB doit être possible !
