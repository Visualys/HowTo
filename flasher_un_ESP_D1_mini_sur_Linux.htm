1. Installation de esptools

Lancer la ligne de commande : 
sudo apt install esptool
Télécharger la dernière release de espeasy-mega.
ex : ESP_Easy_mega_20210223_normal_ESP8266_4M1M.bin
renommer le fichier téléchargé en esp.bin (pour simplier la ligne de commande)

2. Flasher l'ESP D1 mini

Connecter le D1 mini au port USB.
Lancer la ligne de commande : 
esptool --port /dev/ttyUSB0 --baud 115200 write_flash -fm dio -fs 32m 0x00000 esp.bin

3. Réglages de l'ESP D1 mini

Au démarrage, le D1 mini est réglé en mode AP (il crée son propre réseau wifi).
Il faut se connecter à son wifi (nommé ESP_Easy par défaut) avec le mot de passe : configesp
une fois connecté, il faut utiliser un explorateur web et se rendre à l'adresse 192.168.4.1/setup
La page affichée permet de pouvoir renseigner le réseau (wifi & mot de passe) auquel le D1 Mini doit se connecter.
