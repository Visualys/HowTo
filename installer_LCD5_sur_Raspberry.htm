1. Carte SD

Créer une carte SD (en utilisant par exemple Raspberry Imager)
Avant de l'insérer dans le raspberry, modifier le fichier config.txt
ajouter à la fin du fichier :
max_usb_current=1
hdmi_group=2
hdmi_mode=87
hdmi_cvt 800 480 60 6 0 0 0
hdmi_drive=1
Insérer la carte dans le raspberry et connecter le LCD5 au raspberry.
L'alimentation ne doit pas être connectée au raspberry lui-même mais sur le LCD (qui alimentera le raspberry). puis démarrer l'installation

2. Activer la fonction tactile

Pour utiliser la fonction tactile du LCD, dans une console raspberry, lancer les commandes suivantes :
git clone https://github.com/waveshare/LCD-show.git
cd LCD-show
chmod +x LCD5-show
./LCD5-show
