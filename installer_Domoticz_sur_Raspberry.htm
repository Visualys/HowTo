1. Installation de Domoticz

Sur le raspberry, lancer la ligne de commande : 
curl - L install.domoticz.com | bash
Utiliser le port 8080 (par défaut). Pour se connecter à domoticz sur ce même raspberry, lancer l'explorateur web à l'adresse : localhost:8080.
Pour se connecter à partir d'un autre appareil connecté sur le même réseau, il faut taper l'adresse ip du raspberry suivi par ":port" (ex : 192.168.1.4:8080.
